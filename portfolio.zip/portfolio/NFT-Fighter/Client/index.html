This site can gather the NFT's from two wallet's. Pick 3 from each and prepare to fight! My code will randomly generate the NFT's you pick with battle stats where they will face off against each other in a style similar to rumble royale!

Don't have a wallet to use? Click on Random Draw for it to select some NFT's from the database for you to battle!

Notes: by using this App you are storing versions of your NFT's in its database. This app is currently only set for the ETH blockchain.
